# tcsv

```
NAME:
   tcsv - transpose stdin and join by comma

USAGE:
   tcsv [global options] command [command options] [arguments...]

VERSION:
   0.0.1

COMMANDS:
     help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --separator value, -s value  separator to join strings (default: ",")
   --help, -h                   show help
   --version, -v                print the version
```
